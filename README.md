# Product Management Super-Skill

Comprehensive product management skill merging **Perplexity Computer's 6 PM skills** with **Claude Code's product team skills**.

## What's Inside

| Section | Description |
|---------|-------------|
| Gap Analysis | Capability comparison matrix |
| Feature Specs & PRDs | 4 PRD formats, scope management, review cycles |
| Roadmap & Prioritization | RICE, MoSCoW, ICE, Now/Next/Later, quarterly themes |
| Sprint & Agile | 5 ceremony formats, velocity forecasting, health scoring |
| Metrics & OKRs | North Star metrics, L1/L2 hierarchy, dashboard principles |
| User Research & UX | Method selection, interview analysis, personas, journey mapping |
| UI Design Systems | Token categories, atomic design, responsive breakpoints, WCAG |
| Competitive Analysis | Feature matrices, win/loss methodology, monitoring |
| Stakeholder Comms | 4 audience templates, ROAM risk, ADR format |
| Persistent Planning | Three-file planning system, 3-Strike Error Protocol |
| Cross-Skill Workflows | End-to-end feature dev, quarterly planning, research sprints |

## Sources Merged

**Perplexity Computer:** pm-feature-spec, pm-roadmap-management, pm-metrics-tracking, pm-competitive-analysis, pm-stakeholder-comms, pm-user-research-synthesis

**Claude Code:** PM Toolkit (RICE + Interview Analyzer), Agile Product Owner, UX Researcher Designer, UI Design System, Scrum Master, Planning with Files

## Usage
Upload the `SKILL.md` file to your Perplexity Computer user settings or Claude Code skills directory.
